# KK.PasswordManager

**Локальный менеджер паролей с акцентом на безопасность и приватность**

KK.PasswordManager — это локальное приложение для управления паролями, которое хранит ваши данные исключительно на вашем компьютере. Все данные шифруются с использованием вашего PIN-кода, что обеспечивает полный контроль над вашей информацией.

## ⚠️ Важное предупреждение

**PIN-код является ключом ко всем вашим паролям.** Если вы потеряете или забудете PIN-код, вы **навсегда потеряете доступ** ко всем сохраненным паролям. Приложение не имеет бэкдоров, мастер-паролей или функций восстановления — это осознанный выбор в пользу безопасности.

## 🚀 Возможности

- 🔒 **Локальное хранение** — все данные хранятся только на вашем устройстве
- 🛡️ **PIN-шифрование** — все пароли шифруются с использованием вашего PIN-кода
- 👤 **Простая аутентификация** — быстрый вход с помощью PIN
- 📁 **Структурированное хранение** — поиск паролей по названиям и сайтам
- 🎯 **Простой интерфейс** — минималистичный дизайн для удобства использования
- 💾 **Автономная работа** — не требует интернет-соединения

## 📋 Требования к системе

- Windows (x64 или x86)
- 200MB свободного места

## 📥 Установка

1. Скачайте последнюю версию из раздела [Releases](https://github.com/KavvaiiKira/KK.PasswordManager/releases)
2. Распакуйте архив в удобное место
3. Запустите `KK.PasswordManager.exe`

## 🏁 Начало работы

### Первый запуск

1. При первом запуске создайте нового пользователя
2. Введите имя пользователя (только для отображения)
3. **ВНИМАТЕЛЬНО** выберите и введите ваш PIN-код

### Структура записи пароля

Каждая запись пароля содержит:
- **Name** — название/описание пароля (например, "Рабочая почта")
- **Site** — веб-сайт или приложение (например, "example.com")
- **UserName** — имя пользователя, email или логин
- **Password** — пароль (будет зашифрован)

### Рекомендации по безопасности

- 🚫 **Не используйте простые PIN-коды** (1234, 0000, даты рождения)
- 🔑 **Выберите надежный PIN-код** длиной не менее 6 цифр
- 💾 **Регулярно делайте резервные копии** файла с сохраненными в зашифрованном виде паролями (*папка_приложения*/kkbin/passwords.json)
- 📝 **Запишите PIN-код в надежном месте** (не на компьютере!)

---
